Desarrollo de Sitio Web de Adminitracion Bibliotecaria

"Periodo:2021-Actualidad"

En este proyecto se desarrolla en la Universidad de Cundinamarca como proyecto de gestion de conocimiento de la carrera de Ingenieria de Sistemas y computacion en el cual se identifico potencial en el desarrollo de un Sitio web para la gestion de los documentos y operaciones que se realizan en la biblioteca publica del municipio de Simijaca, en la cual se tiene como resultado Una interfaz grafica creada en Html y CSS del Login de dicha paginas ademas de que este ayudo a la comprension de las importancias de las bibliotecas como enormes bases de datos que guardan no solo texto si no informacion cultural muy importante

Desarrollo de Software contable para agricultores

"Periodo:2023-Actualidad"

Este software contable para agricultores no esta tan avandzado pero aun asi esta diseñada para ser una herramienta diseñada específicamente para ayudar a las empresas agrícolas a gestionar sus finanzas. Este tipo de software ofrece una amplia gama de funciones que pueden ayudar a los agricultores a realizar un seguimiento de sus ingresos y gastos, administrar su inventario, generar informes financieros y tomar decisiones más informadas sobre su negocio.
